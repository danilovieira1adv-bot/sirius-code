REQUISITOS
──────────
- Python 3.9 ou superior
- pip (gerenciador de pacotes Python)
- Conexão com internet
- Chave de API Sirius Code (obtenha em siriusopen.com.br)

INSTALAÇÃO
──────────

1. Instale o Sirius Code:

   pip install sirius-code

   No Linux/Mac, se precisar de permissão:
   pip install sirius-code --user

2. Configure sua chave de API:

   sirius config --key SUA_CHAVE_AQUI

3. Configure a URL do servidor (já vem configurada):

   sirius config --url https://code.siriusopen.com.br

4. Verifique a conexão:

   sirius status

   Resposta esperada: ✓ Sirius Code online


COMO USAR
─────────

MODO INTERATIVO (recomendado):

   sirius

   Abre o terminal interativo. Digite sua tarefa e pressione Enter.
   Use /help para ver os comandos disponíveis.

MODO DIRETO (uma tarefa):

   sirius "crie uma API REST em FastAPI"
   sirius "refatore o arquivo app.py para usar async"
   sirius "crie testes unitários para meu projeto"
   sirius "analise os erros no arquivo main.py"
   sirius "crie um Dockerfile para este projeto"


EXEMPLOS PRÁTICOS
─────────────────

   # Criar uma API completa
   sirius "crie uma API REST com autenticação JWT em FastAPI"

COMANDOS DO TERMINAL INTERATIVO
────────────────────────────────

   /help      Mostrar ajuda
   /clear     Limpar a tela
   /session   Ver ID da sessão atual
   /exit      Sair


CONFIGURAÇÕES
─────────────

   # Ver configurações atuais
   sirius config --show

   # Trocar chave de API
   sirius config --key NOVA_CHAVE

   # Verificar status
   sirius status


SOLUÇÃO DE PROBLEMAS
────────────────────

Erro "API key inválida":
→ Verifique sua chave: sirius config --show
→ Reconfigure: sirius config --key SUA_CHAVE

Erro "Créditos insuficientes":
→ Acesse siriusopen.com.br para recarregar

Erro de conexão:
→ Verifique sua internet
→ Tente: sirius config --url https://code.siriusopen.com.br

O Sirius não encontrou o arquivo:
→ Execute o sirius no diretório do seu projeto
→ Use caminhos absolutos: sirius "edite /caminho/completo/arquivo.py"
